core.config_loader module¶

class core.config_loader.JSONConfigLoader[source]¶

Loads a set of tasks and a schedule for them from a JSON file:

{
  "tasks":
  {
    "<task_id>": {
        "type": "<task_class>",
    },
    "<task_id>": {
        "type": "<task_class>",
        "world": "<world_id>"
    }
    "...": "..."
  },
  "worlds":
  {
    "<world_id>": {
      "type": "<world_class>",
    }
  },
  "scheduler":
    {
      "type": "<scheduler_class>",
      "args": {
          "<scheduler_arg>": <scheduler_arg_value>,
        }
    }
}

The scheduler scheduler_arg_value could be a container including task ids, which will be replaced by the concrete tasks instances.
